The Project

The team of Swagfish are developing a huge video game (the next big thing) in the most fashionable way there is! How do they do it? Read on and find out..

Automatic builds?

You know, laziness makes a good programmer. The system is set up with a script that builds the whole project.

They just love testing!

When the build script is run, all sorts of tests follow ("is the grid correct?", "is the player allowed to do that?"). Lovely stuff!

Always running

The newest version of the game is always available. Just head out to www.example.com and check it out. Their automatic build never fails to amaze.

The coolest strategy

It's modern, it's agile and it's fun. The Test Driven Development is the process that the Swagfish dedicate their work to. When there's a new feature, it will be managed on a new feature branch and everything is protected. (The code is loosely coupled and follows good object oriented design practices)

Continuously awesome

If you weren't impressed yet, this is the part where you get blown away. Travis makes sure all code is integrated into a shared repository and everything is kept track of.
